About these units

Bits vs Bytes

Bit (b) is the smallest unit: one 0 or 1. Byte (B) = 8 bits. One byte can represent one character (e.g. in ASCII). File sizes and storage are usually given in bytes (B, KB, MB). Network and link speeds are often given in bits per second (Mbps, Gbps) — so “100 Mbps” means 100 megabit/s, i.e. 12.5 MB/s.

Octets vs Bytes

Octet means exactly 8 bits, so 1 octet = 1 byte. The term is common in French (where “octet” is the usual word for byte) and in networking and telecom: RFCs and standards often use “octet” to avoid ambiguity, because on some older systems a “byte” could be 6, 7, or 9 bits.

On modern systems, byte and octet are the same. So the scaled units match too: 1 megabyte (MB) = 1 megaoctet (Mo), 1 gigabyte (GB) = 1 gigaoctet (Go), 1 kilobyte (KB) = 1 kilooctet (Ko), and so on. “Mo” and “Go” are the usual French abbreviations; in this converter, octets are in the same list as bytes. The converter treats bytes and octets identically.

Decimal (SI) vs Binary (IEC) — MB vs MiB

Decimal units (KB, MB, GB, TB) use powers of 1,000: 1 KB = 1,000 B, 1 MB = 1,000,000 B. Used by storage manufacturers (so “1 TB” looks larger) and in many apps and docs.

Binary units (KiB, MiB, GiB, TiB) use powers of 1,024: 1 KiB = 1,024 B, 1 MiB = 1,048,576 B. Better match for how computers address memory (powers of 2). Used in RAM, by many OS tools (e.g. ls -lh), and in technical contexts. So “1 GB” (decimal) = 1 billion bytes; “1 GiB” (binary) = 1,073,741,824 bytes — about 7.4% more.
